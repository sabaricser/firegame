### Local Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sabaricser/firegame.git

   ```

2. **Install dependencies**:

### Controls
- **Arrow Keys**: Move your ship
- **Space**: Shoot
- **P**: Pause
- **ESC**: Exit
